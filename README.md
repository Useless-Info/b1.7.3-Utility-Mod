# b1.7.3-Utility-Mod

This is a Minecraft jar mod for beta 1.7.3 that I made for myself because I wanted some quality of life things like zoom, and I doubt there's many mods for this version. 

## Features
* Zoom (Default bind is 'x')
* Zoom out in F5 (Use '=' and '-' keys)
* Durability HUD for your armor and held tool (You can change the position in video settings)

I will probably add more stuff every so often, but it'll be decently sparse.

## Installation
This doesn't use a mod loader like you'd be used to. Instead you add files to the Minecraft jar itself.

### No custom launcher or Betacraft
1. Locate your Minecraft b1.7.3 jar
2. Open the jar with WinRAR, 7z, or whatever you want
3. Delete the file named "META-INF"
4. Copy the .class files from this repo into the Minecraft jar (replace the files if it ask you about duplicates)

### Prism
1. Click on your beta 1.7.3 instance and click "edit" on the right
2. Click on "version" on the left
3. Click "Add to Minecraft.jar" on the right
4. Select the folder containing the .class files from here
